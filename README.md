# pdl-c
Un repositorio de pdl en c

Necesitas tener Make instalado, y crear un alias que sea: 
alias make='make -B', desde /pdl-c haces make, de esta forma compilas.
para ejectutar ./out, las flags de compilación de gcc son: -Wall -Wextra.
en /bin van los binarios, pero make se asegura de compilarlo todo en ./out.

Te recomiendo ejecutar siempre desde la terminal en linux, de no hacerlo podrías
tener diversos errores que no controlo.
